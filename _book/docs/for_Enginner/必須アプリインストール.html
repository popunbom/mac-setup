必須アプリインストール

必須アプリインストール

CotEditor − 軽量テキストエディタ

  • macOS 標準の「テキストエディタ」は癖があるため、オススメしない

    • 標準でリッチテキスト入力であるということ

  • App Store からインストール

iTerm − ターミナルソフト

  • macOS 標準のターミナルより機能が豊富なので、オススメ

Visual Studio Code − エディタ

ToyViewer − 画像処理アプリケーション

  • 簡単な画像処理が行えるので、オススメ

  • App Store からインストール

MacTeX − macOS 用 LaTeX ディストリビューション

  • macOS で LaTeX を書くために必要

  • MacTeX.pkg をクリックしてダウンロード
